## Project Overview

**StudySit** is a modern, clean, and community-focused static website designed for a coworking space. Its core mission is to provide a warm, productive, and inspiring environment for students, creators, and freelancers.

The site is built to be the digital storefront, clearly communicating the space's ethos, available facilities, pricing plans, and contact information. It is designed with a focus on **clean aesthetics, fast load times, and intuitive navigation**â€”hallmarks of a great user experience.

##  Features

The website is structured around clear, dedicated pages to guide potential members:

* **About Page:** Details the vision, mission, and community ethos of StudySit.
* **Spaces Page:** Showcases the various areas available (quiet zones, collaborative tables, meeting rooms).
* **Pricing Page:** Clearly outlines different membership tiers (hourly, daily, monthly) and associated benefits.
* **Contact Page:** Provides location details, operating hours, and a form for inquiries.
* **Clean and Responsive Design:** Ensures a seamless experience on all devices (desktop, tablet, and mobile).
* **Clear Navigation:** Intuitive and easy-to-use navigation bar for rapid access to all key information.

## Getting Started

Follow these simple steps to view and run the StudySit website locally on your machine.

### Prerequisites

You only need a modern web browser installed on your computer.

### Local Setup

The project requires **no server configuration** or complex build tools.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/DEBMUKHERJEE15/StudySit.git](https://github.com/DEBMUKHERJEE15/StudySit.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd StudySit
    ```

3.  **Run the application:**
    * Simply locate the **`index.html`** file in the root of the project directory.
    * Right-click on `index.html` and choose **"Open with"** -> **[Your Web Browser]** (e.g., Chrome, Firefox).

The website will immediately open in your browser, ready for testing and development!
